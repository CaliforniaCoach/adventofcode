(2) y1 + t1 * dy1 = y2 + t2 * dy2 (= y)

(1a) t1 = (x2 - x1 + t2 * dx2) / dx1
(2a) t1 = (y2 - y1 + t2 * dy2) / dy1

t2 = ((x1 - x2) * dy1 - (y1 - y2) * dx1) / (dx2 * dy1 - dy2 * dx1)
